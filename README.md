# ETEM-H2-CMM

Interactive visualisations for the article *"Hydrogen Technology Pathways in
Metropolitan Energy Systems: An Application to Montréal."*

This repository hosts the interactive HTML figures produced with the
**ETEM-H₂-CMM** model — a linear energy-system optimization model analysing
hydrogen decarbonization pathways for the Communauté métropolitaine de
Montréal (CMM) toward carbon neutrality by 2050.

## Overview

ETEM-H₂-CMM downscales GHG shares across the 82 municipalities of the CMM and
evaluates hydrogen technology pathways under different decarbonization scenarios
(GHG1, GHG2, GHG-Max). The figures in this repository present the model's
scenario results as interactive plots.
